# Bracket Demo

The Bracket Demo is a built-in sample project with a bracket body, Aluminum 6061 material, fixed mounting-hole supports, a top-face load, generated mesh summary, result artifacts, and report output.

It can be loaded as a static stress study or as a dynamic structural study from the sample picker. Static results show stress, displacement, and safety factor. Dynamic results add transient frame metadata for playback where result frames are available.
