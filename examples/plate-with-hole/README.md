# Beam Demo

This example directory is named `plate-with-hole`, but the current `plate` sample id presents as the Beam Demo in the app.

The Beam Demo models an end-loaded beam with a payload mass on the free end. It supports static and dynamic sample loading, selectable fixed/load faces, generated mesh summaries, local solver results, dynamic playback where available, and report artifacts.

For static local solves, the solver service can route this sample through the Euler-Bernoulli beam path to produce deterministic bending stress and displacement estimates.
